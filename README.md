Project Title

Facial Emotion Recognition Using Convolutional Neural Networks

Overview

This project builds an AI system that detects and classifies human facial emotions from images and real-time webcam feeds using deep learning and computer vision techniques.

Emotions Detected

Happy

Sad

Angry

Fear

Surprise

Neutral

Key Features

Face detection using OpenCV

Image preprocessing and normalization

CNN-based emotion classification

Real-time emotion recognition via webcam
